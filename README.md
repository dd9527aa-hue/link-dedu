# 🔗 智能链接去重系统

一个功能强大的多用户链接去重工具，支持推特ID和群组链接的独立管理。

## ✨ 功能特性

- 🏷️ **双主库系统**：推特ID和群组链接独立管理
- 👥 **多用户支持**：用户注册登录，数据完全隔离
- 🔍 **智能去重**：多种去重模式，精确查找新增内容
- 📊 **数据管理**：导入、导出、批量操作
- 🎨 **美观界面**：响应式设计，现代UI

## 🚀 快速开始

1. 访问：https://您的用户名.github.io/link-dedupe-system/
2. 注册新账号（第一个用户自动成为管理员）
3. 开始管理您的链接数据

## 💡 使用说明

### 主库管理
- 分别管理推特ID和群组链接
- 支持文本导入和文件上传
- 实时统计显示

### 去重对比
- 输入新链接与主库对比
- 智能识别重复内容
- 批量添加到主库

### 数据导出
- 支持纯文本、JSON、CSV、HTML格式
- 可以选择性导出部分数据
- 一键复制到剪贴板

## 🔧 技术栈

- HTML5 + CSS3 + JavaScript
- 本地存储（localStorage）
- Font Awesome 图标
- 响应式设计

## 📱 移动端支持

完美适配手机、平板和桌面设备。
